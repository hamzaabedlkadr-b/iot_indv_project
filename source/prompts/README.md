# LLM Prompt Log

The assignment requires documenting LLM usage.

This folder now includes:

- [`prompt_log_template.md`](./prompt_log_template.md) as the structure for recording meaningful prompts,
- notes about prompts used during implementation,
- prompt refinements,
- reflections on code quality,
- strengths and limitations of the LLM workflow.

Recommended workflow:

1. Copy the template when a prompt leads to meaningful code or design changes.
2. Record the exact prompt.
3. Note what was accepted, corrected, or rejected.
4. Summarize what you learned from that interaction.
